

# Containers Build

```
docker build -t frontend-react ./frontend-react-js
```

# Containers Run

```
docker run --rm -p 3000:3000 -it frontend-react

```


# Containers Show

```
docker ps
```

# Container Stop

```
docker stop <id>
```

# Docker Logs

```
docker logs aws-cloud-project-bootcamp-frontend-react-js-1
```
